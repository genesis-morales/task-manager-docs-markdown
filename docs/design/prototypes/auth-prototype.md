### US-UX-1.5 – Interactive prototype
Stitch AI prototype:
- Registration → login → app flow
- Password reset flow
- Basic profile edit flow

### Required Screens

- Registration (US-UX-1.1)
- Login (US-UX-1.2)
- Forgot password (US-UX-1.3a)
- Email sent / Check your email (US-UX-1.3b)
- New password + success (US-UX-1.3c)
- Profile & Settings (US-UX-1.4)
- Dashboard (Epic 0, existing)

---

### Logical User Flows (for documentation)

1. **Registration flow**
   - Registration → Registration success message → Login → Dashboard

2. **Login flow**
   - Login (valid credentials) → Dashboard  
   - Login (invalid) → Show error → Stay on Login

3. **Password reset flow**
   - Login → “Forgot password?” → Forgot password screen  
   - Forgot password → Email sent / Check your email  
   - From email link → New password screen → Success → Login

4. **Profile flow**
   - Dashboard → Profile & Settings  
   - Profile & Settings → Edit profile → Save → Profile & Settings (updated)
