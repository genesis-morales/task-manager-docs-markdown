## How to Use

1. **Access the prototype** - Open the Stitch AI link above
2. **Navigate** - Click through interactive elements
3. **Review components** - Check UI Kit for component specifications
4. **Provide feedback** - Use Stitch AI comments feature
5. **Test interactions** - Try button hovers, card clicks, etc.

## Feedback Process

### For Designers
- Use Stitch AI built-in commenting
- Tag team members with @mentions
- Reference specific screens or components

### For Developers
- Review component specifications in UI Kit
- Check hover/active states
- Verify spacing and colors against design tokens
- Open GitHub issues for implementation questions

### For Stakeholders
- Use comments for high-level feedback
- Schedule walkthrough sessions if needed
- Request changes via GitHub issues
